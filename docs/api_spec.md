# API Specification

## add(a, b)
Description: Add two numbers and return the result.
Inputs: a, b
Output: number

## divide(a, b)
Description: Divide a by b and return the result.
Inputs: a, b
Output: number
Raises: ZeroDivisionError

## greet(name)
Description: Return a greeting string.
Inputs: name
Output: string

## process_items(items)
Description: Process list of items and return processed list.
Inputs: items (list)
Output: list
